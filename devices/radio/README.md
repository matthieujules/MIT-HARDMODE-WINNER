# Radio

A stationary audio device that plays music and speaks. No motors, no camera, no mic. Pure sound output.

## Sensors

None.

## Actions

| Action | Description | Example |
|--------|-------------|---------|
| `speak` | Say something via TTS | `speak("Dinner is ready")` |
| `play_music` | Play music or ambient audio by genre | `play_music("lo-fi")` |

## Notes

- USB speaker output.
- Handles both TTS speech and music playback.
